# 年金機構 電子通知書 届出単位変換ツール

日本年金機構の電子通知書（XML）を、届出（到達番号）単位の一覧表形式に変換し、PDFで保存するツールです。

---

## 使い方（3ステップ）

1. **index.html** をダブルクリックしてブラウザで開く
2. 画面の指示に従い、**① 通知書フォルダ** と **② RenkiStylesheet フォルダ** を選択
3. **変換する** をクリック → 表示されたら **印刷 / PDF保存** でPDFに保存

※ すでに.xslを置き換えた「変換フォルダ」をお持ちの場合は、①のみでOKです。

---

## 配布について

**このフォルダをそのまま渡すだけで使えます。**

- ZIPに圧縮してメール添付や共有フォルダで渡してOK
- 受け取った人はZIPを解凍し、**index.html をダブルクリック**するだけ
- インストール不要・オフラインで動作・データは外部に送信されません

詳細は **配布要領.md** をご覧ください。

---

## フォルダ構成

```
renkin-converter/
├── index.html          ← これを開く
├── README.md           ← このファイル
├── 配布要領.md         ← 配布者向け
└── RenkiStylesheet/    ← 必ず含める（届出単位用XSL）
    └── *.xsl（13ファイル）
```

---

## 注意事項

- 外字を含む電子通知書は変換できません（日本年金機構の仕様）
- 推奨ブラウザ：Chrome、Edge、Safari
