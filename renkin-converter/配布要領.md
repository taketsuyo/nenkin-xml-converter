# 届出単位変換ツール 配布要領

## 配布するファイル

### 推奨：単一ファイルで配布

**「届出単位変換ツール.html」を1ファイル渡すだけでOKです。**

```
渡すもの：
  届出単位変換ツール.html  ← これだけ（約380KB）
```

- 変換に必要なスタイルシートはファイル内に組み込み済み
- フォルダ構成を気にする必要なし
- 受け取った人はダブルクリックするだけ

---

### 代替：フォルダごと配布

単一ファイルが使えない場合は、従来の `index.html` + `RenkiStylesheet` フォルダをまとめて渡すことも可能です。

---

## 配布方法

| 方法 | 手順 |
|------|------|
| **メール添付** | 「届出単位変換ツール.html」をそのまま添付（380KBなので問題なし） |
| **共有フォルダ** | 共有フォルダに置いてパスを案内 |
| **USBメモリ** | ファイルをコピーして渡す |
| **クラウド共有** | OneDrive / Google Drive 等にアップロードしてリンク共有 |

---

## 受け取った人への案内文（コピー用）

```
【届出単位変換ツール】

年金機構の電子通知書を一覧表形式に変換し、PDF保存するツールです。

■ 使い方
1. 「届出単位変換ツール.html」をダブルクリック（ブラウザで開きます）
2. 通知書が入ったフォルダを選択
3. 「変換する」をクリック
4. 「印刷 / PDF保存」でPDFに保存

■ 安心してお使いください
・インストール不要です
・インターネット接続は不要です（オフラインで動作）
・ファイルは外部に送信されません（すべてパソコン内で処理）
・推奨ブラウザ：Chrome、Edge
```

## よくある質問

| 質問 | 回答 |
|------|------|
| インストールは必要？ | 不要。ファイルをダブルクリックするだけ |
| インターネットは必要？ | 不要。オフラインで動作 |
| データは外部に送信される？ | いいえ。すべてPC内で処理 |
| Windows / Mac 両方使える？ | はい |
| ファイル名を変えてもいい？ | はい。自由に変えてOK |
| 複数人に配布してよい？ | はい |

---

## トラブル時の対応

| 症状 | 対処 |
|------|------|
| ファイルがメモ帳で開く | 右クリック →「プログラムから開く」→ Chrome / Edge を選択 |
| フォルダを選べない | 「フォルダの選択」ダイアログで、フォルダをクリックして「アップロード」を押す |
| 「通知書が見つからない」と表示 | XMLファイルが入ったフォルダを選んでいるか確認 |
| 印刷ボタンが反応しない | ブラウザを Chrome または Edge に変更 |
