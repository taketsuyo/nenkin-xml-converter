# 年金機構 電子通知書 届出単位変換ツール

日本年金機構の電子通知書（XML）を、届出（到達番号）単位の一覧表形式に変換し、PDFで保存できるツールです。

**インストール不要・オフライン動作・データは外部に送信されません。**

---

## クイックスタート

| 利用形態 | ファイル | 特徴 |
|----------|----------|------|
| **おすすめ（1ステップ）** | [届出単位変換ツール.html](renkin-converter/届出単位変換ツール.html) | 通知書フォルダを選ぶだけ。XSL内蔵（約380KB） |
| **2ステップ版** | [index.html](renkin-converter/index.html) | 通知書フォルダと RenkiStylesheet フォルダをそれぞれ選択 |

### 使い方（3ステップ）

1. **届出単位変換ツール.html** をダブルクリックしてブラウザで開く
2. 画面の指示に従い、**通知書が入ったフォルダ**を選択
3. **「変換する」** をクリック → 表示されたら **「印刷 / PDF保存」** でPDFに保存

## 注意事項

- **外字を含む電子通知書**は変換できません（日本年金機構の仕様による制限）
- **推奨ブラウザ**：Google Chrome、Microsoft Edge、Safari
- 変換前に通知書フォルダを**複製**してから操作することを推奨します

---

## RenkiStylesheet について

本ツールは、日本年金機構が提供する **RenkiStylesheet**（届出単位表示用XSLスタイルシート）に準拠しています。  
XSLファイルは日本年金機構の仕様に基づくもので、変換ロジックの根拠となっています。
